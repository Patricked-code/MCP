# MCP_PERMISSIONS_MODEL.md

## Rôle
Modèle de permissions MCP.

## Niveaux
- read_only
- documentation_write
- scoped_code_write
- deploy_controlled
- admin_human_only

## Règles
Les permissions doivent être explicites, minimales, traçables et révocables. Toute permission dangereuse demande validation humaine.
