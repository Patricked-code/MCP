# GitHub Codespaces — MCP Onboarding Engine

Ce dépôt est configuré pour être ouvert dans GitHub Codespaces.

## Objectif

Fournir un environnement prêt pour développer, tester et documenter le MCP WealthTech.

## Organisation cible

`chainsolutions-wealthtech`

## Règles

- Ne jamais exposer de secrets.
- Ne jamais committer `.env`, `secrets/`, `keys/`, dumps ou clés privées.
- Travailler sur branche dédiée.
- Passer par Pull Request pour les modifications importantes.
- Documenter toute évolution dans `memory/`.

## Registre MCP

Sur le serveur MCP : `/app/data/mcp-git-registry.json`

Dans Codespaces : `/workspaces/MCP/data/mcp-git-registry.json`
