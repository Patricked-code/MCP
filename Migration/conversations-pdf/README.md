# Conversations PDF

Ce repertoire est le point d entree cible pour les exports PDF complets des conversations utiles au projet `chainsolutions-wealthtech`.

Regles:

- conserver les exports complets quand ils sont exploitables comme preuve historique;
- ne pas ajouter de secret, token, cle privee ou dump sensible;
- preferer un nom date et stable;
- lier chaque PDF important depuis `Migration/index/`.

Les anciens exports deja presents dans `Migration/PDF/` restent conserves; ce dossier sert de structure cible normalisee pour les prochaines integrations.
