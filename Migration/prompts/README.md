# Prompts

Ce repertoire regroupe les prompts de pilotage et consignes reutilisables.

Chaque prompt ajoute ici doit indiquer:

- son objectif;
- l agent cible;
- les entrees attendues;
- les limites de securite;
- les fichiers ou routes concernes.

Les prompts ne doivent jamais contenir de token, mot de passe, cle privee ou chemin serveur sensible non generalise.
