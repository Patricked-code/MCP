# Decisions

Ce repertoire garde les decisions de migration et de gouvernance.

Chaque entree devrait contenir:

- la date;
- le contexte;
- la decision;
- les consequences;
- les liens vers conversations, documents ou PRs;
- les risques ouverts.

Ce dossier sert a eviter la perte de contexte entre ChatGPT, Claude, Codex, le MCP et les interventions humaines.
