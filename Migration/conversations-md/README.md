# Conversations Markdown

Ce repertoire contient les versions Markdown exploitables par les agents IA.

Objectif:

- transformer les conversations utiles en sources lisibles par ChatGPT, Claude, Codex et les agents MCP;
- extraire les decisions, contraintes, plans, commandes et points de reprise;
- garder les fichiers courts, indexes et citables depuis `Migration/index/`.

Aucun secret brut ne doit etre recopie depuis une conversation.
