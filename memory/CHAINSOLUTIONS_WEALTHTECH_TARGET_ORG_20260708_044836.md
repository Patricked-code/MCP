# Paramétrage MCP — chainsolutions-wealthtech

`chainsolutions-wealthtech` est l'organisation cible principale du MCP.

## Décision

Tous les nouveaux dépôts GitHub créés par le MCP doivent aller dans `chainsolutions-wealthtech`.

## Comptes connus

- `chainsolutions-wealthtech` : organisation cible principale.
- `Patricked-code` : compte source actuel contenant MCP, api, Front, Stablecoin, Investbourse et autres repos.
- `Wealthtechinnovations` : compte secondaire à connecter via le token présent sur S2.

## Règles

- Créer les nouveaux repos dans `chainsolutions-wealthtech`.
- Écrire par branche + Pull Request en priorité.
- Ne jamais afficher ni committer token, `.env`, clé privée, PEM, dump ou secret.
- Migrer ou mirrorer progressivement les repos `Patricked-code/*` vers `chainsolutions-wealthtech/*`.
