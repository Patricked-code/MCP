## Sécurité

Ne jamais stocker dans ce dossier :

- tokens Bearer ;
- clés privées ;
- `.env` complets ;
- API keys ;
- mots de passe ;
- dumps SQL ;
- sauvegardes serveur ;
- secrets relayer.

Les adresses publiques, hashes de transaction, chemins de logs et commits Git peuvent être conservés pour traçabilité.
