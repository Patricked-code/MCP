# PROJECT_BRIEF.md

## Role
Brief strategique du MCP WealthTech.

## Projet
Le MCP WealthTech SSH Bridge sert de couche d'acces controle, de memoire operationnelle et de gouvernance entre ChatGPT, Claude Code, Codex, GitHub et les serveurs.

## Objectif
Eviter la perte de contexte, le codage a l'aveugle, les regressions et les confusions entre serveurs, depots, branches, domaines et agents.

## Utilisateurs

Humains opérateurs, responsables techniques, ChatGPT, Claude, Codex et agents MCP autorisés.

## Résultats attendus

Un inventaire fiable, un point de reprise complet, des permissions explicites, des mappings vérifiés, une documentation parent/enfant, des outils bornés et un workflow branche/PR sans secret.

## Indicateurs de réussite

Couverture documentaire obligatoire, cohérence GitHub/S1, JSON `.mcp` valides, tâches traçables, absence de secret détecté, tests documentés et onboarding projet fondé sur des preuves.

## Hors objectif implicite

Ce brief n’autorise aucun déploiement, migration, suppression, modification de production ou accès à un secret.
